# Galaxy Conquest - Warp Portal & Base Building Guide

## 🌀 Warp Portal System

### How Warp Portals Work
- **Spawn Locations**: Portals appear at specific distance checkpoints during combat
- **Checkpoint Distances**: 5km, 10km, 15km, 25km, 35km, 50km, 75km, 100km, 150km, 200km
- **Portal Tiers**: Different colors and capabilities based on distance
  - **Common** (5-9km): Green portals - Basic outposts
  - **Uncommon** (10-24km): Blue portals - Standard bases  
  - **Rare** (25-49km): Purple portals - Major outposts
  - **Epic** (50-99km): Orange portals - Advanced bases
  - **Legendary** (100km+): Pink portals - Ultimate strongholds

### Using Warp Portals
1. **Detection**: Portals appear as swirling energy anomalies with distance markers
2. **Activation**: Fly your ship into the portal to trigger entry sequence
3. **Confirmation**: Choose whether to enter base building mode or continue combat
4. **Access**: Opens the Galaxy Base Builder interface at that checkpoint

## 🏗️ Base Building System

### Building Categories

#### 🏛️ Infrastructure
- **Power Generator**: Produces energy for base operations
- **Command Center**: Increases base efficiency and unlocks advanced buildings
- **Shield Generator**: Provides defensive protection (requires 10km+)

#### ⚙️ Production
- **Metal Refinery**: Processes metal from raw materials
- **Crystal Harvester**: Extracts rare crystals (requires 15km+)
- **Dark Matter Collector**: Harvests exotic dark matter (requires 50km+)

#### 🚀 Military  
- **Defense Turret**: Automated base defense (requires 5km+)
- **Missile Silo**: Long-range area attack system (requires 25km+)
- **Ship Dock**: Vessel maintenance and upgrade facility (requires 10km+)

#### 🔬 Research
- **Research Lab**: Develops new technologies (requires 15km+)
- **Quantum Computer**: Accelerates research processes (requires 35km+)
- **Observatory**: Scans for discoveries and resources (requires 25km+)

### Base Building Interface
- **Grid System**: 8x6 building slots per base
- **Resource Display**: Real-time tracking of Energy, Metal, Crystals, Dark Matter
- **Building Browser**: Category-filtered available structures
- **Upgrade System**: Multi-level building progression (Lv.1-10)
- **Statistics Panel**: Production rates, defense rating, research bonuses

## 💎 Galaxy Resource System

### Resource Types
1. **⚡ Energy**: Powers base operations and building construction
2. **🔧 Metal**: Primary construction material for most buildings
3. **💎 Crystals**: Rare materials for advanced technologies
4. **🌌 Dark Matter**: Exotic resource for legendary upgrades

### Resource Collection
- **Combat Drops**: Defeat enemies to collect resources
  - Regular enemies: Small amounts of Energy and Metal
  - Mini-bosses: Moderate amounts including Crystals
  - Major bosses: Large amounts including rare Dark Matter
- **Base Production**: Idle resource generation from constructed buildings
- **Tier Bonuses**: Higher tier bases multiply production rates
- **Special Events**: Random resource discovery opportunities

### Resource Management
- **Capacity Limits**: Each resource has maximum storage capacity
- **Conversion System**: Trade resources at different exchange rates
- **Save Integration**: Resources persist between game sessions
- **Cross-Session**: Collect idle production when returning to game

## 🎮 Gameplay Integration

### Combat to Base Building Flow
1. **Start Combat**: Begin space shooter gameplay
2. **Progress Distance**: Advance through enemy waves
3. **Reach Checkpoint**: Achieve milestone distances (5km, 10km, etc.)
4. **Portal Appearance**: Warp portals spawn at checkpoint distances
5. **Portal Entry**: Choose to enter base building mode
6. **Base Construction**: Build and upgrade facilities
7. **Return to Combat**: Resume shooting gameplay with base benefits

### Strategic Elements
- **Resource Planning**: Balance immediate combat needs vs. long-term base development
- **Checkpoint Strategy**: Decide which checkpoints to develop as major bases
- **Production Chains**: Optimize building combinations for maximum efficiency  
- **Defense Investment**: Build military structures to protect resource investments
- **Research Focus**: Prioritize technology development for gameplay advantages

## 🌟 Advanced Features

### Idle Mining System
- **Background Production**: Resources generate even when not playing
- **Production Rates**: Based on constructed buildings and base tier
- **Automatic Collection**: Gather idle resources when returning to game
- **Compound Growth**: Multiple bases provide cumulative benefits

### Base Tier Bonuses
- **Tier Multipliers**: Higher tier bases provide production bonuses
  - Common: 1.0x base production
  - Uncommon: 1.2x production multiplier
  - Rare: 1.5x production multiplier  
  - Epic: 2.0x production multiplier
  - Legendary: 3.0x production multiplier

### Strategic Depth
- **Base Network**: Manage multiple bases across different checkpoints
- **Specialization**: Focus bases on production, military, or research
- **Resource Flow**: Plan resource distribution between bases
- **Long-term Planning**: Build toward ultimate late-game objectives

## 🚀 Quick Start Guide

1. **Play the Game**: Start combat and reach 5km distance
2. **Find Portal**: Look for green swirling warp portal
3. **Enter Portal**: Fly into portal and confirm entry
4. **Build Base**: Start with Power Generator and Metal Refinery
5. **Expand**: Add more buildings as resources allow
6. **Return**: Go back to combat to collect more resources
7. **Repeat**: Continue cycle of combat → resources → building → upgrades

**Pro Tip**: Focus on production buildings first to establish resource generation, then add military and research facilities for advanced capabilities!
