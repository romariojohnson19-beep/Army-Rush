## 📞 Technical Support

### Common Questions

**Q: How do I adjust music volume?**
A: Edit `js/audio-manager.js` and change `this.musicVolume = 0.4`

**Q: How do I adjust sound effect volume?**
A: Edit `js/audio-manager.js` and change `this.sfxVolume = 0.5`

**Q: Can I add more music tracks?**
A: Yes! Add file to `assets/music/` and update track list in audio manager

**Q: Can I add different weapon sounds?**
A: Yes! Add files to `assets/sounds/` and create new play methods

**Q: Will this affect mobile performance?**
A: No! Audio has 0% FPS impact on any device
